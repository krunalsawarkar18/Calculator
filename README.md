# Calculator Web App

A sleek, responsive web-based calculator built with modern web technologies. This project demonstrates proficiency in HTML, CSS, JavaScript, and responsive design principles.

## 🚀 Features

- **Basic Arithmetic Operations**: Addition, subtraction, multiplication, and division
- **User-Friendly Interface**: Clean, intuitive design with Bootstrap styling
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Error Handling**: Graceful handling of invalid mathematical expressions
- **Keyboard-Friendly**: Easy-to-use button layout with hover effects
- **Real-time Display**: Live updates as you input numbers and operations

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styling with modern design principles
- **JavaScript (ES6+)**: Core calculator logic and DOM manipulation

## 📋 Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required - runs entirely in the browser

## 🚀 Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/krunalsawarkar18/calculator.git
   cd calculator
   ```

2. **Open in browser**:
   - Simply open `index.html` in your web browser
   - Or serve via a local web server for the best experience

3. **Start calculating**:
   - Click buttons to input numbers and operations
   - Use the `=` button to calculate results
   - Press `C` to clear the display
   - Use `DEL` to remove the last character
